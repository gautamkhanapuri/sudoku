# Sudoku Puzzles -  Front end for Sudoku Solver to solve and generate.
This has been written as a single file with plain javascript and CSS. No frameworks used.

Manually add any puzzle from any source and solve it
Load puzzles from 81 length strings.
Generate puzzles based on the difficulty.

Additional Functionalities:
Enable hints which provides list of valid values by rowm column and box elimination.
Enable guess to help backtrack. 1 level of guess is added.
Difficulty ranges from 4-52, higher the number, higher the difficulty.
